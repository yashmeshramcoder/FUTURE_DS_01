# 📊 Business Sales Dashboard (Tableau)

## 📌 Project Overview

This project is a simple and interactive **Business Sales Dashboard** created using Tableau. It helps in understanding how sales are performing over time, which products are doing well, and how sales are distributed across different countries.

## 🎯 Objectives

* To track monthly sales trends
* To identify top-selling products
* To analyze sales across different countries

## 📊 Key Insights

* Sales show a steady growth trend, with the highest peak in November
* A few top products contribute a large share of total sales
* Sales are mainly concentrated in specific countries

## 🛠 Tools Used

* Tableau for data visualization

## 📂 Files Included

* `Business_Sales_Dashboard.twb` → Tableau dashboard file
* `Screenshot (160).png` → Dashboard preview image

## 🚀 How to Use

1. Download the `.twb` file
2. Open it using Tableau
3. Interact with the dashboard to explore insights
